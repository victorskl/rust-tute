# dlq_analysis

Contain multiple binaries

### Clean and Build

```bash
cargo clean
cargo build
```

### Run

```bash
./target/debug/dlq_list
./target/debug/dlq_alt
./target/debug/dlq_analysis ../.nogit/DLQ/
```

### Run with Cargo

```bash
cargo run --bin dlq_analysis ../.nogit/DLQ/
```

```bash
cargo run ../.nogit/DLQ/
```
